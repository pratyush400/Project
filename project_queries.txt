Author: Pratyush K. Jha & Daniel Macha

Simple queries 
1. Determine the number of bird observations in each neighbourhood, list the ten neighbourhoods with the highest number of birds, and order the results alphabetically by neighbourhood name.
2. How many species of birds are in [neighborhood name]
3. Retrieve a list of birds observed on 6th Feb, 2022
4. Retrieve a list of tree species that contains “flower”
Junction table query
5. For a specified nest year, list each bird species and the number of different trees in which that species built nests.
6. For a specified tree, list all birds that nested in and years it occurred 
Update queries
7. Add new bird species names to the database. (Species names)
8. Add a new bird observation linked to that species, including birdspeciesid, observation date and neighborhood
9. Add a new nesting record that connects a bird to a tree for a specific year.
10.Add a new tree to the database
